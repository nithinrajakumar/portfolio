Scratch Project

September 8, 2017

Project Objective:

Create a game or story where the user must use the keyboard to fundamentally affect what happens.

In my Computer Science course, I had to work with Nick Sacca to create this creative game on Scratch. We wanted to create an interactive game with a ball getting from point a to point b. During the project, I coded most of the game, and learned how to use variables to make specific commands activate and deactivate. One particular obstacle that I had trouble coding was the jumping mechanism used in the game. I solved that problem by using the help from another developer, and then using their idea to solve the problem in my own way. Throughout the course of the project, my partner and I recorded our progress in a project notebook.
